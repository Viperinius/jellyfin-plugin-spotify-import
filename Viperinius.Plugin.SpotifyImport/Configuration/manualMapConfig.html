Spotify Import - Manual Track Map

Put provider tracks in here to force mapping it to a specific Jellyfin track.

After adding a new row, replace the dummy text with the ID of the target Jellyfin track item in one of the following formats:

  • Raw ID (ex.: bfd9b42542e752bfdd63617e7966ba6a)
  • "Copy Stream URI" result (ex.: http://<JELLYFIN IP>/Items/bfd9b42542e752bfdd63617e7966ba6a/Download?api_key=...)
  • URL from web UI (ex.: http://<JELLYFIN IP>/web/?#/details?id=bfd9b42542e752bfdd63617e7966ba6a&serverId=...)
Then confirm the ID; it should get parsed and display the track name of that item.

You can use the different "new entry" buttons for choosing different input formats of the Spotify track information:

  • "Raw": Specify the Spotify track using its Spotify ID or the name, album name and artists
  • "JSON": Specify the Spotify track using the JSON entry of a missing tracks file
As long as a Spotify ID is given, the other Spotify fields are ignored.
